## spring mvc 게시판

### 사용기술 
* java, jsp, javascript, css
* bootstrap4, jQuery , Ajax, jstl, el 
* spring, springJDBC
* RDBMS - mysql

### 개발환경
* 이클립스
* spring 4.3.5
* maven 3.5
* jdk 1.8
* tomcat 8.5
